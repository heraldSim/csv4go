# CSV4Go

CSV4Go is a library written in pure Go to use csv data more comfortable

### Suported Go version
- golang >= 1.13

### Installation

```bash
go get github.com/heraldSim/csv4go
```

### Feature

- Support Collection model
- Include Map, Filter, Reduce
- All cell data is converting Go-Map
